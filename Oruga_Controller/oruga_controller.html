Oruga Controller

Cómo usar

  • Mantén presionada una flecha para mover: avanzar, retroceder, izquierda, derecha.
  • Al soltar, se envía Stop automáticamente para detener el movimiento.
  • Ajusta los sliders (0–255) y pulsa “Confirmar velocidades” para aplicar ambos lados.
  • Los cambios se envían vía Orion; abre “Ver logs” para diagnóstico si algo falla.
